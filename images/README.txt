LETAKKAN GAMBAR ANDA DI FOLDER INI (images/)
=============================================
Nama file harus PERSIS seperti di bawah (huruf kecil semua, format .jpg).
Rasio ideal 16:9 (mis. 1200x675 px atau 1600x900 px).

THUMBNAIL + GAMBAR UTAMA TIAP ARTIKEL:
  pajak-marketplace.jpg     -> Artikel "Lautan Transaksi yang Tak Terlihat" (Pajak/PMK 37)
  transaksi-ecommerce.jpg   -> Gambar SISIPAN di tengah artikel pajak (grafik/foto e-commerce)
  dana-darurat.jpg          -> Artikel "Dana Darurat: Bantal yang Sering Dilupakan"
  bunga-majemuk.jpg         -> Artikel "Bunga Majemuk: Keajaiban yang Butuh Kesabaran"
  inflasi.jpg               -> Artikel "Memahami Inflasi"
  spt-tahunan.jpg           -> Artikel "SPT Tahunan Tanpa Drama"
  lifestyle-inflation.jpg   -> Artikel "Lifestyle Inflation"

CARA KERJA:
- Selama file belum ada, blog menampilkan placeholder gradien berlabel nama file.
- Begitu Anda taruh file dengan nama yang benar, gambar otomatis muncul
  (di thumbnail beranda, gambar utama artikel, dan sisipan).
- Jika ingin pakai .png, ganti juga nama di index.html (cari "images/").
